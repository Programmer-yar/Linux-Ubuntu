-> Install a package using command line:
	sudo dpkg -i <package_name.deb>
	
-> shutdown/restart using Terminal:
	systemctl poweroff
	systemctl reboot
	
	Use button combo:
	Ctrl+Alt+Del
	
-> zip unzip:
	- sudo apt install zip unzip
	- command: "unzip <file_name>.zip"
			   "zip -r <new_name>.zip <folder/file_name>"
	

-> "export" command:
	- used to export 'environment variables' to child processes
	- View all exported variables of current shell:
	  export -p
	- set up an environment variable
	  export <variable_name> = <value>
